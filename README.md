# line-flex-sample
FlexMessageのサンプル

# インストール

```
$ npm install
```

# 起動

```
$ node index.js
```

# ngrokインストール

```
$ npm i -g ngrok
```

# ngrok起動

```
$ ngrok http 3000
```

# serveoの場合

```
$ npx miso-develop/serveo hoge 3000
```
